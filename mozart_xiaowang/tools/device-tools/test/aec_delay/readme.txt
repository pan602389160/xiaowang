1.程序功能：
  测试回声延时,适用于canna平台oss或alsa音频系统
2.测试要求：
  canna板接好speaker，接好mic，确保硬件能够录音放音
  speaker距离数字麦克风10cm左右，测试的这20秒内不要有人大声说话
  压力测试的时候,输入aecstop命令后不要说话，输入之前可以说话
  条件允许的话可以找个安静环境测试。
3.测试方法：
  (1)kill 系统中录音和放音相关进程
  (2)将test拷贝到sd卡，再将sd卡插入canna板

4.测试步骤:
 进入x1000系统，
  cd /mnt/sdcard/test
  chmod 777 run.sh
  如果不进行压力测试,短时间看到测试结果，执行以下命令
  ./run.sh 0
  如果进行压力测试,长时间拷机
  ./run.sh 1
  压力测试模式在test目录键盘输入aecstop，20秒后结束测试，
  不输入的时候一直处于ａｅｃ录音模式

５.测试结果:
  测试结束后程序会有以下打印：
--------------------------------------
root@SmartAudio:/mnt/sdcard/aaa# ./run.sh 0
===========test start==============
system mplayer successful
===========test finish=============
start compute_delay_time
amic_offset :10220 dmic_offset :10223
---------------------------------------
-----time play-dmic is :0.188 (ms)--
---------------------------------------
其中
0.188就是回声延时，表示回声延时是0.188毫秒
理论上回声延时值是0-1毫秒是正常的，大于1毫秒或者是负值属于异常情况。
